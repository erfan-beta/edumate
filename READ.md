# 🎓 EduMate Backend

Spring Boot Backend for EduMate E-Learning Platform.

## 🚀 Features
- JWT Authentication (Access + Refresh Tokens)
- Role-Based Access Control (Student, Instructor, Admin)
- PostgreSQL Database
- Spring Security

## 🛠️ Tech Stack
- Java 17
- Spring Boot 3.2
- PostgreSQL
- Maven

## 📦 Setup
1. Clone repo
2. Configure `application.properties`
3. Run `mvn spring-boot:run`
